MEMBER SUBS

Setautocomplete(enabled as boolean) When True, entries are autocompleted as typed
Setclearonreturn(enabled as boolean) When True the current entry is cleared when return is pressed
Setmaxentrues(value as integer) Sets the maximum number of entries in the list
about() Shows the about form
Clearentries() Clears all entries in the list
Cropentries(tomaxentries as boolean,Optional value as integer) Shortens the list. When tomaxentries the list is shortened to the value set in Setmaxentries, value is only used when tomaxentries is False, it enables you to define the ammount of entries you want to shorten the list to.
Loadentries(filename as string) Loads all list entries from a give filename, 1 item per line
Setcleartip(tip as string) Sets the tooltiptext for the clear button
Setcombotip(tip as string) Sets the tooltiptext for the combo box
Setcurrententry(entry as string) Sets the current entry in the list to entry
Saveentries(filename as string) Saves all list entries to a given filename, can be loaded again with Loadentries

MEMBER FUNCTIONS

Getmaxentries() as integer Returns the value set with Setmaxentries
Addentry(entry as string) as boolean Adds an entry to the list, True is returned if the item was added. The item will not be added if it is either empty or allready in the list.
Delentrystring(entry as string) as boolean Automatically finds a given entry by string and removes it from the list, if the deletion is successful True is returned.
Delentryindex(index as integer) as boolean Deletes an entry by its index, entries start at 0.
Findentry(entry as string,completematch as boolean) as integer Finds an entry in the list by string and returns the entries Index. If unsuccessful -1 is returned.
Getcurrententry() as string Returns the current entry as a string. The current entry is the one being typed.
Getentry(index as integer) as string Returns the entry defined by index.
Getnoofentries() as integer Gets the number of entries in the list.

EVENTS

entryadded(entry as string) Raised when an item is added to the list, the item is returned in entry.
entrynotadded(entry as string) Raised when an item is not added, because it allready exists in the list or Setmaxentries is set to 0. The item is returned in entry.
clearrequest() Raised when the clear button is clicked
entrychanged(entry as string) Raised when data is typed into the combo box, the new item is returned in entry.
